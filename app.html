Agenda de Citas

Aplicación de escritorio desarrollada en C# y .NET para la gestión de hospitales: doctores, pacientes y citas.

GitHub

Sobre el Proyecto

La aplicación incluye módulos de administración de pacientes, doctores y recepcionistas, con control de accesos por roles. Además, permite generar reportes en PDF y consultar expedientes clínicos.
